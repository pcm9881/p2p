# 세금신고
- 비영업대금의 이자소득 27.5% (소득세 + 지방세)
소득세: 수익 * 25% 의 원단위 절사 
지방세: 소득세 * 10% 의 의 원단위 절사
*원단위절사: 10원 미만의 세금은 절사
